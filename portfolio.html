Projeto 1

Quiz

Esse projeto foi Desenvolvido por para obter a nota da Materia de POO
_ Programação Orientada a Objeto _ esse projeto tem como objetivo
criar um quiz com perguntas e respostas para jogar com os amigos.
Esse quiz tem um menu,onde o jogador pode adiciona seu nome e
iniciar uma nova partida assim como ir para o Ranking de Pontuação.
Dentro do do jogo apareceram as perguntas e a cada pergunta respondida
o sistema verificará se o usuário respondeu certo ou errado e dará uma
pontuação dependendo se a resposta for certa ou errada e entao passará para
a proxima questão até o fim onde retornará para o menu inicial.
No Ranking de pontuação o sistema listará as 10 pessoas com as melhores
pontuações.

Projeto 2

Saber Idade

Esse projeto foi desenvolvido para realizar alguns teste de Programação Web Coding.
O objetivo é verificar as idades dos usuários: eles informam o ano de
seu nascimento, selecionam o gênero e o sistema calcula automaticamente
a idade e a faixa etária correspondente.
