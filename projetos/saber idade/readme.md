### Verificador de Idade
Este é um projeto simples de verificação de idade e gênero, desenvolvido usando HTML, CSS e JavaScript.

### Funcionalidades
O usuário insere o ano de nascimento e seleciona o gênero (masculino ou feminino).
Ao clicar no botão "Verificar Idade", o programa calcula a idade do usuário e exibe uma mensagem com a classificação de idade (criança, jovem, adulto ou idoso) e o gênero correspondente.
Uma imagem é exibida de acordo com a classificação de idade e gênero do usuário.
Estrutura do Projeto

### O projeto é composto por três arquivos:
saberidade.html: Arquivo HTML que contém a estrutura da página.
saberidade.css: Arquivo CSS que define o estilo da página.
saberidade.js: Arquivo JavaScript que contém a lógica de funcionamento do verificador de idade.

### Como Usar
Abra o arquivo saberidade.html em um navegador web.
Insira o ano de nascimento no campo correspondente.
Selecione o gênero (masculino ou feminino).
Clique no botão "Verificar Idade".
O resultado será exibido na tela, com a classificação de idade e gênero, juntamente com a imagem correspondente.
